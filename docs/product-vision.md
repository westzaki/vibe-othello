# Product Vision

## Overview

Vibe Othello は、小学校高学年〜中学生のオセロ初心者が、楽しく対局しながら「ちゃんと勝てるようになる」ことを目指すブラウザゲームである。

プレイヤーは、1P / 2P 対局、CPU 対戦、パス通知、ふりかえり、レビュー局面からの練習を通じて、角・危険手・合法手数・終盤判断といった基本戦略を自然に身につける。

このアプリの価値は、単に対局できることではない。
「なぜ勝てたか / なぜ負けたか」を理解し、次の対局で再現できる強さを少しずつ身につけられることにある。

## Current Product Direction

当面の方針は、大きな追加機能や新しい演出要素を前提にせず、現在の UX をブラウザゲームとして十分に信頼できる品質へ引き上げることである。

優先するもの：

- 盤面が見やすく、操作しやすい
- 1P / 2P / ふりかえり / 練習の流れが迷わない
- 重要なイベントが自然に伝わる
- 音やアニメーションが気持ちよい
- コピーがやさしく、短く、前向きである
- モバイルでも破綻しない
- プロトタイプ感ではなく、完成した軽いゲームとして触れる

名前付きの案内役や大きな演出要素は、現時点ではプロダクトの中心に置かない。
必要になった場合も、まずは現行 UX の品質を損なわない小さな表現として検討する。

## Emotional Core

このアプリは、「あと一手で勝てたのに」という悔しさを、「次は勝てそう」という期待に変える体験を提供する。

プレイヤーが感じてほしいこと：

- 置ける場所が分かる
- 何を考えればよいか少し見えてくる
- 負けても、次に試すことがある
- 勝ったときに、良かった判断を再現できそうだと思える
- もう一局やってみたいと思える

体験の中心は、盤面・手触り・短いフィードバック・練習導線である。

## Core Experience

### 1. Start

Start 画面は、設定フォームではなく、対局前の入口として機能する。

重視すること：

- 1P / 2P の選択が分かりやすい
- CPU の強さが直感的に選べる
- 黒 / 白の選択が迷わない
- スタートボタンが最も明確な行動である
- 温かく、ゲームらしい見た目である

### 2. Game

Game 画面では、盤面が常に主役である。

重視すること：

- 現在の番が分かる
- 合法手が見つけやすい
- パスや結果などの重要イベントに気づける
- スコアや流れは補助情報として見える
- 棋譜や分析情報は初心者の集中を邪魔しない
- 石を置く、返る、音が鳴る体験が気持ちよい

### 3. Result

結果画面では、勝敗をはっきり伝えつつ、次の行動へつなげる。

重視すること：

- 1P では「あなたの勝ち / あなたの負け / 引き分け」として分かる
- 2P では「黒の勝ち / 白の勝ち / 引き分け」として分かる
- 最終盤面が見える
- ふりかえり、もう一回、タイトルへ戻る導線が分かりやすい
- 勝っても負けても前向きな気持ちで終われる

### 4. Review

Review は、全手を分析するレポートではなく、次に活かせる局面だけを見る学習体験である。

負け / 引き分けでは、勝負どころと練習導線を中心にする。
勝った試合では、反省会ではなく「勝てた理由を見てみよう」として、良い判断を再現しやすくする。

### 5. Practice

Practice は、レビューで見つけた重要局面からすぐ試せる再挑戦の場である。

説明を読んで終わるのではなく、別の手を打って「次は勝てる」を体験することを目的にする。

## Tone & Feel

- やさしい
- 温かい
- 軽い
- ちょっと楽しい
- 子どもっぽすぎない
- 分析ツールではなく、考えるゲーム
- 負けても嫌な気持ちにならない
- 文章は短く、前向きで、押し付けない

## Product Quality Bar

今後の改善では、以下をプロダクト品質の基準とする。

- 主要画面で横スクロールや表示崩れがない
- 320px 前後のモバイル幅でも操作できる
- ボタンと非インタラクティブな表示が見分けられる
- 主要アクションが一目で分かる
- UI の色、影、角丸、音、アニメーションに一貫性がある
- 状態変化が急すぎず、待たされすぎない
- 文言が画面目的に合っている
- 新しい要素が盤面の見やすさを邪魔しない

## Differentiation

- 初心者向けに情報を絞った対局体験
- 全手解析ではなく、重要局面だけを見るレビュー
- レビューから練習へすぐ移れる導線
- 「たくさん取る」から「相手にいい手をあげない」へ自然に進む学習設計
- ブラウザで気軽に遊べるが、手触りと見た目は完成品として気持ちよい

## Retention Design

継続したくなる理由：

- 勝てるようになる実感がある
- もう一局が軽い
- 操作や音が気持ちいい
- 負けたあとに試すことが分かる
- 勝ったあとに良かった判断を再現したくなる

## Learning Philosophy

プレイヤーは知識を暗記するのではなく、対局・ふりかえり・練習を通じて、勝てる考え方を少しずつ身につけていく。

## Non-goals

- プロレベルの高度な解析
- 数値中心の難解な評価表示
- 重い講義形式の学習体験
- 名前付きの案内役を前提にした体験設計
- 派手な演出で盤面への集中を奪うこと
